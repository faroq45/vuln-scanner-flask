# Installation:
- git clone https://github.com/krishpranav/vuln-scanner-flask
- cd vuln-scanner-flask
- sudo chmod +x *

# Dependencies:
```
make sure you have redis-cli, python3, pip3
```

# Steps To Run This:
```
python3 -m venv env
source env/bin/activate
bash start.sh
```

- now head to http://0.0.0.0:8080 and start your scan
