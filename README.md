# Installation:
- git clone https://github.com/krishpranav/vuln-scanner-flask
- cd vuln-scanner-flask
- sudo chmod +x *


# Steps To Run This:
- python3 -m venv env
- source env/bin/activate
- python3 -m pip install -r requirements.txt
- python3 app.py

- Now Go To http://127.0.0.1:5000/ and head your scan
